CHANGE CHARACTERS

Hiding a game will remove every character from that game from your tierlist! Don't click unless you are sure you want to change! You cannot undo this.

The sections appended with (Extras) contain mugshots for returning characters with different appearances. If you don't want the tier list to contain duplicate characters or are wary of spoilers, don't check them.

Note that most of the Extras sections have major spoilers for their games!

An expansive tierlist maker for the Ace Attorney series.

Dock Tierlist allows you to always see the tierlist, even when scrolling down.

Hide Controls removes the tool widget and arrows from the tierlist, allowing for a simpler design if no customization is neccessary.

Change Characters allows you to add and remove characters to the selection that you want to tier.

Filter is a way to only feature specific characters to your selection of characters that you want to tier.

Save/Load is a handy way to generate the layout of your tierlist as text, which can be stored and re-used later by pasting it in the open textarea.

Reset removes every character from the tierlist and puts it back to the default settings.

Change Layout is an optional button that moves the characters from below the tierlist to right of it for easy access. This change can be reverted.

FILTER

Choosing any filter will hide all unselected characters that don't match the chosen filters. Display every character again by unchecking every filter.

For best results, toggle on the Extras section for the games you want to filter with. Beware of spoilers for games you haven't completed when using the filters!
